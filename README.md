# Installation
```sh
pip install -r requirements.txt
./chia_la init-db
```

# Usage'
```sh
chia_la.py [OPTIONS] COMMAND [ARGS]...
```
where COMMAND:
- analyze
- drop-db
- init-db
- stat

# Adding logs
```
chia_la.py analyze -d <some_log_dir>
```
# Getting statistic
```sh
chia_la.py stat
```
More options information you can get with --help option
